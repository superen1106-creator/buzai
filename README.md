# 不在

一个“口袋里的避难所”网页 App。

## 体验方式

[打开「不在」](https://superen1106-creator.github.io/buzai/)

## 1.1「房间开始记得你」

- 首次使用先从“想静止、想流动、想被包住、想换一种存在”进入，仍可查看全部元素
- 回访时直接回到上一次的房间，也可以换一种存在
- 每次到访和写信都会留下极轻的自然痕迹，不打卡、不排名
- 每类元素拥有触摸、划过、生长、发光或重量感互动
- 回应可以选择“只听见我”或“陪我想一会儿”
- 停留不显示倒计时，可选择一段呼吸、睡前、情绪之后或只是回来
- 不在场证明支持安全分享，朋友可直达对应元素，私密来信不会进入分享
- 游客写下第二封信后，才自然提示跨设备保存
- 支持添加到手机桌面和基础离线打开

## 基础体验

- 从“在吗？”进入“不在”
- 从 25 种自然元素中选择此刻的化身，或写下自己的元素
- 每种元素拥有独立的几何、线条或粒子动态
- 通过“闭上眼 · 深呼吸”，在避难所里停留
- 和自己说说话，把文字写给所选的元素
- 在“我的来信”里回看曾经写下的文字，也可以二次确认后删除
- 生成并保存“不在场证明”
- 25 种元素拥有各自的“不在场证明”文字
- 没有倒计时的停留模式，以及随元素变化的粉红／棕色声音纹理（风声、雨声、溪流声、海浪声、林间声等，不是音乐）
- 返回元素选择时，正在播放的声音会自动关闭
- 游客模式无需注册，来信保存在当前设备
- 邮箱登录后，化身与来信会在手机、电脑之间同步

游客内容只保存在浏览器本地。登录用户的数据保存在 Supabase，数据库开启了行级权限，每位用户只能读取和修改自己的数据。
